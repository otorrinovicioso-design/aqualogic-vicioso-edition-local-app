# AquaLogic Vicioso Edition

Aplicación local para la gestión de acuarios y núcleos zoológicos.

## Características
- Gestión de Censo y Población.
- Seguimiento de Parámetros de Agua con gráficos.
- Bitácora Sanitaria.
- Módulos de Reproducción y Alimentación.
- **100% Local**: Los datos se guardan en el dispositivo (LocalStorage).
- **IA AquaGenius**: Consultas inteligentes integradas.

## Requisitos
- Node.js
- Gemini API Key (opcional para IA)
